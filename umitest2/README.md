[![NPM downloads](http://img.shields.io/npm/dm/umitest2.svg?style=flat)](https://npmjs.org/package/umitest2)



## Install

```bash
# or yarn
$ npm install
```

```bash
$ npm run build --watch
$ npm run start
```

## Usage

Configure in `.umirc.js`,

```js
export default {
  plugins: [
    ['umi-plugin-umitest2', options],
  ],
}
```
